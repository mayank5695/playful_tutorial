
- Download playful executable from playful.is.tuebingen.mpg.de

- Make sure it is executable. In a terminal:

        chmod +x ./playful

- Copy the executable to the folder in which this README file is located

- Execute the hello world program, in a terminal:

        ./playful execute  	  

- See all the files involved in the program:

        ./playful trail

- See the code corresponding to print_hello node:

        ./playful word print_hello
